Quant Finance Research Projects
Quant Project · ML × Quantitative Finance
Regime-Conditioned KAN
Trading Strategy

Most ML models applied to markets assume the market behaves the same way all the time. This project challenges that: first detect which market regime you're in using a Hidden Markov Model, then route to a specialist forecaster (a Kolmogorov-Arnold Network) trained only on days that looked like today. KANs replace fixed activation functions with learnable spline curves, making the model's logic directly readable. The result is a transparent, regime-aware system that reveals cross-asset relationships invisible to single-model approaches.

Regime Detection
Unsupervised HMM on SPY, QQQ, GLD, TLT features discovers genuine market structure — post-2020, 2022 rate cycle, and 2013–2019 bull run each emerge as distinct regimes without labeled data.
Interpretability
KAN spline plots are a direct window into model logic. In calm bull regimes, bond momentum outpredicts equity momentum — a cross-asset signal only visible with regime specialization.
Honest Methodology
Walk-forward validation, zero data leakage. An initial backtest produced 6.28 Sharpe — caught, diagnosed as leakage, corrected. Underperformance vs benchmark honestly reported.
